Instructions

App is hosted on dxvish.github.io

Since github.io runs on HTTPS, calling the Provided API throws an error as its hosted on HTTP. For running the app without issues, please follow the instructions below

  • Open dxvish.github.io
  • Choose From Date and To Date
  • Click on Fetch Button
  • On the URL bar, chrome will throw an warning stating that the site is trying to load unsafe script(Mixed content - Both HTTPS & HTTP requests)
  • Click on the Red error botton on URL bar
  • Toggle - Allow Unsafe Scripts
  • Chose the date and click on fetch
  • Toggle between Table and Chart View using the control on the dialog boxs
